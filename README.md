 🌍 Tour and Travel Management System

 📌 Project Overview

The **Tour and Travel Management System** is a web-based application developed using **PHP and MySQL**.
It helps users explore travel packages, make bookings, and manage their trips easily.
The system also provides an admin panel to manage packages, users, and bookings efficiently.

---

 🎯 Features

 👤 User Module

* User Registration & Login
* View Travel Packages
* Book Tour Packages
* View Booking History

 🛠️ Admin Module

* Add / Update / Delete Packages
* Manage Users
* Manage Bookings
* Generate Reports

---

 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP

---

 💻 System Requirements

### Hardware:

* Minimum 4GB RAM
* Intel i3 or above

 Software:

* XAMPP
* Web Browser (Chrome/Edge)
* Code Editor (VS Code)

---

 🚀 Installation Steps

1. Install XAMPP
2. Start **Apache** and **MySQL**
3. Copy project folder to:

   ```
   htdocs/
   ```
4. Import database in **phpMyAdmin**
5. Open browser and run:

   ```
   http://localhost/Tour-and-Travel-Management-System
   ```
